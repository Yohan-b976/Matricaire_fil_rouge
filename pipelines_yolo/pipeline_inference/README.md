Pour faire fonctionner la pipeline d'inférence, il faut :
-   créer un environnement virtuel à l'aide du fichier requirements.txt;
-   remplir le fichier data;
-   modifier les chemins relatifs des fichiers et dossiers nécessaires à la fin du script_inference.py,
-   à la fin du script_inference.py, choisir Visualize = True (enregitrement de la photo traitée avec prédictions des fleurs sous format jpg dans le dossier results) ou False.

C'est bon, vous pouvez désormais lancer le script depuis votre terminal, de la façon suivante !
python script_inference.py
